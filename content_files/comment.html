Comments

Comments help the programmer to understand the program. Let's say there is programmer A who wrote some 100 to 500 lines of code to perform some task. Programmer B has to analyze the code written by programmer A. B may take some time to analyze the code. And he can understand it in one day or two days based on the complexity of the code. What if the number of lines in the code is more than 1000 to 5000? Then, it will take more than a week to understand the code for B. But, using comments in a code and writing what happens in the code will help programmer B to understand the code written by programmer A much faster.

It's as if we have to travel long distances in many directions. It's difficult for us to remember the route to our destination. But, if we have some landmarks in mid-way. They will help us to reach our destination much faster.

Example 3 : Single line comment

if(5 > 6):
    print("hello") # Single line comment

    print("hello") # Single line comment

Writing comments will not effect the code at all because they will be ignored by the compiler.
