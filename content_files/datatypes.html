Datatypes

An empty box can be filled with any object (books, toys, cloths, etc). Datatype means what type of data (object) is placed in the variable (box)

Datatype tell what type of data we are using the code to perform the operations

Python has following built-in datatypes :

Text Type : str

Numeric Type : int, float, complex

Sequence Type : list, tuple, range

Mapping Type : dict

Set Type : set, frozenset

Boolean Type : bool

Binary Type : bytes, bytearray, memoryview

None Type : NoneType



Creating and Assigining :

Example 16 : Datatypes

a = "helloworld" # string
a = 5 # integer
a = 5.0 # float
a = 2j # complex
a = ["one", "two", "three"] # list
a = ("one", "two", "three") # tuple
a = range(5) # range
a = {"name":"BMW", "number":5} # dict
a = {"one", "two", "number"} # set
a = frozenset({"one", "two", "number"}) # frozenset
a = True # bool
a = b"Hello"  # bytes
a = bytesarray(5)  # bytearray
a = memoryview(bytes(5))  # memoryview

Don't worry about all these datatypes and how they will work. we will look into each one in detail.
