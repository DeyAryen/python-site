Global variables

Global variables are the variables which are created outside the function. So, they can be used inside the function and outside the function.

Example 14 : Globle variable 

a = "!" Global variable
def myFunction():
    print("hello_world " + a) # inside the function

myFunction()
print(a) # outside the function

Output

hello_world !
!

Example 15 : global keyword

x = "outside" # outside
def myFunction():
    global x # global keyword
    x = "inside" # inside

myFunction()
print(x)

Output

inside

Now, try creating the variable inside the function and print it outside the function

Don't worry about the functions that you will understand later in the chapters for now just understand how global variables work.
