Intro long story short

Python, after listening to this word the image we get into our mind is the snake called Python. But, the Python that we are talking about is not a snake it is the programming language used to build many technologies. Why does the Python programming language is invented? In the 1980s, the Python programming language was born at Centrum Wiskunde & Informatica (CWI) in the Netherlands. Guido van Rossum who is working at CWI as a developer for ABC language it is unsuccessful, So he moved on to other programming projects in-them the key one was the amoeba which was a distributed system that was being built collaboratively by CWI. Rossum observed that every application written in the amoeba is either a shell script or a C program that is difficult to understand and takes more time to execute. So, he decided to build a programming language that is easier to use, giving more information in fewer lines. For that, he has to create an interpreter of his own to run python programs. In 1991, the Python programming language became an open source.

Note:
Guido van Rossum, named his invention as Python because he is a big fan of a TV show called Monty Python's Flying Circus.



    Why Python?

  • Python is a general-purpose, object-oriented and high-level programming language.
  • Python is so easy to learn because its syntax is like plain English. So, anyone can learn Python very quickly without any prior programming knowledge.
  • It is versatile, which means it can adapt to many computer-science areas like Machine learning, AI, Data Science, Game Development, Web Development, etc.
  • Python has one of the largest and most supportive developer communities.
  • In Python, we have libraries and frameworks consisting of many pre-defined codes you can reuse in your programs.
  • In some situations, even a small task can take a lot of work in those situations we need to automate things. Using python libraries and frameworks we can create automation scripts that make our work much more automated.
  • Many more.


Code execution process

The python compiler reads a python source code (high-level language). Then it verifies that the instruction is well-formatted, and syntax in Python is mandatory. If there are no errors in the source code compiler will convert the source code into byte code (intermediate language). Byte code then sent to the PVM (Python Virtual Machine). which is a Python interpreter. Converts the byte code into machine code (low-level language).
