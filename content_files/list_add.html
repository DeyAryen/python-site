List - Add

New items can be added to the list using three methods

append(), insert(), extend().


Append method :

append() , Append method will add the item at the end of the list.

Example 25 : Append method

cars = ["BMW", "Benz", "Audi"]
cars.append("KIA")
print(cars)

Output :

["BMW", "Benz", "Audi", "KIA"]

Insert method :

insert() , Using Insert method items can be add any where in the list using indexs.

Example 26 : Insert method

cars = ["BMW", "Benz", "Audi"]
cars.insert(1, "KIA")
print(cars)

Output :

["BMW", "KIA", "Benz", "Audi"]

Note :
The index of the first item in the list is '0'.

Extend method :

extend() , Using Extend method a list can be appended to another list

Example 27 : Extend method

fruits = ["apple", "mango", "banana"]
cars = ["BMW", "Audi", "Benz"]
fruits.extend(cars)
print(fruits)

Output :

 ["apple", "mango", "banana", "BMW", "Audi", "Benz"]

Using extend() method we can also append other user-defined datatypes to list

Example 28 : List and Tuple

fruits = ["apple", "mango", "banana"]  # list type
cars = ("BMW", "Audi", "Benz") # tuple type
fruits.extend(cars)
print(fruits)

Output :

 ["apple", "mango", "banana", "BMW", "Audi", "Benz"]

We will see what is tuple in later chapter.
