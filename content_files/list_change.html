List - Change items

We can change the list items using index number.

Example 35 : change items

fruits = ["apple", "orange", "banana", "pappya", "mango"]       
fruits[3] = "green-apple"
print(fruits)

Output :

["apple", "orange", "banana", "green-apple", "mango"]

Range of items :

To change the range of items in a list we have to provide the starting index, the ending index and the item to replace with.

Example 36 : Range of items

fruits = ["apple", "orange", "banana", "pappya", "mango"]       
fruits[1:3] = ["ORANGE", "BANANA"]
print(fruits)

Output :

["apple", "ORANGE", "BANANA", "pappya", "mango"]

If you insert more items than you replace, the new items will be inserted where you specified, and the remaining items will move accordingly.

Example 37 : Range of items (1)

fruits = ["apple", "orange", "banana", "pappya"]       
fruits[1:2] = ["ORANGE", "BANANA"]
print(fruits)

Output :

["apple", "ORANGE", "BANANA", "banana", "pappya"]

If you insert less items than you replace, the new items will be inserted where you specified, and the remaining items will cleared.

Example 38 : Range of items (2)

fruits = ["apple", "orange", "banana", "pappya", "mango"]       
fruits[1:4] = ["ORANGE"]
print(fruits)

Output :

["apple", "ORANGE"]

NOTE :
The length of the list will change when the number of items inserted does not match the number of items replaced.
