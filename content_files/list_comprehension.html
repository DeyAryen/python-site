List - Comprehension

List comprehension will offer a simple syntax when we are creating a new list from the existing list.

newlist = [expression for item in iterable if condition == True]

Example 49 : Without Comprehension

fruits = ["apple", "kiwi", "orange", "cherry"]
Fruits = []

for x in fruits:
    if "a" in x:
        Fruits.append(x)

print(Fruits)

Output :

["apple", "orange"]

Example 50 : With Comprension

fruits = ["apple", "kiwi", "orange", "cherry"]
Fruits = [x for x in fruits if "a" in x] 
print(Fruits)

Output :

["apple", "orange"]
