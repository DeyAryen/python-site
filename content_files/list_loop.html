list - loop

Looping through a list means accessing item in the list one by one in an order for this we need to know about looping methods in python that are for and while loops. We will discuss about them in later chapters in detail for now just understand how looping is done in lists.

Example 47 : Using for loop

# This code will print the item
# in the list one by one
list = ["apple", "mango", "banana"]
for x in list:
    print(x)

Output :

apple
mango
banana

while loop will be depended on the condition we have given. The code in side the while loop will run until the conditon is True. when the condition become False loop will break.

Example 48 : Using while loop

list = ["apple", "mango", "banana"]
i = 0
while i < len(list):
    print(list[i])
    i = i + 1

Output :

apple
mango
banana
