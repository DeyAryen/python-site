List - Remove

There are four ways to remove items for the list.

remove(), pop(), del, clear()


Remove method :

remove() , In this method we have to provide the item to be removed inside the parentheses of the remove method.

Placing the index values inside the parentheses of the item to remove form the list won't work.

Example 29 : Remove method

cars = ["BMW", "Benz", "Audi"]
cars.remove("Benz")
print(cars)

Output :

["BMW", "Audi"]

Pop method :

pop() , In this method we have to provide index of the respective item to be remove inside the parentheses of the pop method.

Directly providing item inside the parentheses instead of index of that item won't work.

Example 30 : Pop method with index

cars = ["BMW", "Benz", "Audi"]
cars.pop(1)
print(cars)

Output :

["BMW", "Audi"]

In this pop() method, if the index value of the item to be removed is not provided then the last item in the list will be poped out.

Example 31 : Pop method without index

cars = ["BMW", "Benz", "Audi"]
cars.pop()
print(cars)

Output :

["BMW", "Benz"]

del keyword :

del , del is a keyword rather then a function like others.

del keyword can delete the item based on index provided.

Example 32: del keyword

fruits = ["apple", "mango", "banana"]
del fruits[1]
print(fruits)

Output

["apple", "banana"]

del keyword can also delete entire list. If we try to print the list after using the del keyword the complier will rise error.

Example 33 : del keyword to delete list

fruits = ["apple", "mango", "banana"]
del fruits
print(fruits)

Output

    print(thislist)
NameError: name 'thislist' is not defined

Clear method :

clear() , Clear method will clear the list items in the list but won't delete the list.

Example 34 : Clear method

fruits = ["apple", "mango", "banana"]
fruits.clear()
print(fruits)

Output

[]
