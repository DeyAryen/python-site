List - sort

Python has a in-built sort() method for sorting the items inside the list alphanumerically or numerically.

sort() method will prioritizes uppercase letters first and lower case letters next.

If we have 2 words starting with same letter then the sort method will sort the word with second letters of those words.

Example 39 : Sort method

cars = ["BMW", "Benz", "Adui", "bmw", benz, "audi"]
numbers = [8, 4, 6, 7]
cars.sort()
numbers.sort()
print(cars)
print(numbers)

Output :

["Audi", "Benz", "BMW", "audi", "benz", "bmw"]
[4, 6, 7, 8]

Reverse sort :

By defualt the sort method will sort the items inside the list in ascending order. Now, if we want to sort the items in decending order or in reverse order we have to type reverse = True inside the sort method.

Example 40 : Reverse sort method

cars = ["BMW", "Benz", "Adui", "bmw", benz, "audi"]
numbers = [8, 4, 6, 7]
cars.sort(reverse = True)
numbers.sort(reverse = True)
print(cars)
print(numbers)

Output :

["bmw", "benz", "audi", "Audi", "Benz", "Bmw"]
[8, 7, 6, 4]

Note :
sort() method won't work on a list which have both alphanumerically and numerically values.

Case insensitive :

Using key = str.lower will sort the items by ignoreing case sensitivity.

Example 41 : Case insensitive sort

fruits = ["apple", "Orange", "banana", "mango"]
fruits.sort(key = str.lower)
print(fruits)

Output :

["apple", "banana", "mango", "Orange"]

Reverse list :

By using reverse() method we can reverse the positions of item in the list.

Example 42 : Reverse list

fruits = ["apple", "Orange", "banana", "mango"]
fruits.reverse()
print(fruits)

Output :

["mango", "banana", "Orange", "apple"]
