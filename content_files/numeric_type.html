Numeric types

Python has three numeric types :

int float complex

Example 17 : Numeric type

a = 3 # int
b = 3.0 # float
c = 3j # complex

Integer (int) :

In Python, integer can be a whole number, negitive number which do not have any decimal point and the length of the integer has no restriction.

Example 18 : int type

a = 3
a = 3456789345343234
a = -3
a = print(type(3))

Output

class 'int'

Float (float) :

Float number are the numbers which have decimal point. Float can also be scientific numbers with an "e" to indicate the power of 10.

Example 19 : float type

b = 5.0
b = 12E4
b = -34.65e50
b = print(type(5.94))

Output

class 'float'

Complex (complex) :

Complex number is the type of number which has "j" included as an imaginary part.

Example 20 : Complex type

c = j8
c = 2 + j4
c = -j5
c = print(type(j5))

Output

class 'complex'
