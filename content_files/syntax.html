Syntax

Indentation :

Indentation is about how we align the code in each line. In other programming languages indentaion is just about understanding the code it's not mandatory. But, in python indentation plays a important role. Incorrect indentation leads to error or false result.

Example 1 :

if(5 > 6):
    print("hello") # correct
else:
    print("world") # correct

Output

hello

Example 2 :

if(5 > 6):
print("hello") # Incorrect
else:
    print("world") # correct

Output

     print("hello")
    ^
IndentationError: expected an indented block

Basic rule 1 :
Missing indentation (or) Syntax = Missing expected result (or) Geting error.
