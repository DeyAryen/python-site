Variables

Variables are data holding boxes that stores data in them. Creating variables and assigning value to them is very simple in Python compared to other programming languages.

Syntax : variable = value;

In Python, we don't need to specify the datatypes before the variable when we create it. It depends on which type of value we assign to the variable.

Variables can be reassigned just like an item in a box can be replaced.

Example 6 : Creating and Assigning variables

A = 5 # Integer
A = 5.0 # Float
A = '5' # Character
A = "five" # String

Valid variables :

Variables can be short names (like a, b, c) or a more descriptive names (like first_name, age, avg). But, remember variables are case senitive. So, A is different form a.

We have some rules for naming a variables :


1. Should start with letter or underscore.
2. No starting with numbers.
3. Should only contain a-z, A-Z, 0-9, _.
4. Case sensitive A is different form a.

Example 7 : Valid variables 

myvariable = 5 
myvariable2 = 5.0 
_my_variable = '5' 
# best use for multi word variable name
myVariable = "five" # camelCase
MyVariable = 5 # PascalCase
my_variable = 5.0 # snake_case

Example 8 : Invalid variables 

my variable = 5
3myvariable2 = 5.0 
_my-variable = '5'

Mulitple variables - Multiple assign :

We can assign multiple values to multiple variables at a time. Below we have 2 exmples shown that how to assign mulitple values to mulitple variables.

Example 9 : Mulitple assign 

x, y, z = 5, 6, 7
x = y = z = 5
# try runing in your code editor
# print(x)

Casting :

Data type of a variable will be depending on type of value that's assign to it. But, the data type of the variable can be changed by type casting the value.

Example 10 : Conversion

A = int(5) # output 5
B = str(5) # output '5'
C = float(5) # output 5.0

Printing variables :

Using the print() function in Python we can print the values assigned to the variables.

Example 11 : 

First_name = "xyz"
Last_name = "abc"
a = 5
b = "xyz"
Print(First_name, Last_name)
Print(a, b)

Output

xyz abc
9 xyz

Concatenate the values using + operator.
We can concatenate 2 strings but, we cannot concatenate a string and integer.

Example 12 : Concatenation

First_name = "xyz "
Last_name = "abc"
a = 5
b = 4
print(First_name + Last_name)
print(a + b)

Output

xyz abc
9
